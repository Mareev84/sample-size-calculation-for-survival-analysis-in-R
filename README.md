# sample-size-calculation-for-survival-analysis-in-R

В расчётах используется метод Шёнфельда (Schoenfeld, 1983), применяемый для анализа времени до события (лог-ранк тест, модель пропорциональных рисков).

В файле фиксированное наблюдение предполагается, что все пациенты наблюдаются одинаковое время (к примеру полгода), и потери к наблюдению не учитываются.

Во файле равномерный набор учитывается постепенное включение пациентов в течение года, дополнительный период наблюдения и возможные потери (например, 10% в год).

Оба варианта выводят таблицу и график зависимости требуемого размера выборки от ожидаемого эффекта (RRR).
